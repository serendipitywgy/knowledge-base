# 07 · 状态图（State）

← [[06-对象图]] · [[PlantUML从入门到精通|目录]] · 下一章 → [[08-组件图]]

官方：https://plantuml.com/zh/state-diagram

状态图描述对象在生命周期中**处在什么状态、因何事件转移**。订单、工单、连接会话都很适合。

---

## 1. 起止与简单转移

```plantuml
@startuml
[*] --> Draft
Draft --> Submitted : 提交
Submitted --> Approved : 通过
Submitted --> Rejected : 驳回
Rejected --> Draft : 修改
Approved --> [*]
@enduml
```

`[*]`：初始或终止。`状态 --> 状态 : 事件/动作`。

给状态加内部说明：

```plantuml
@startuml
hide empty description
state Submitted
Submitted : entry / 通知审核人
Submitted : do / 等待审批
Submitted --> Approved : 通过
@enduml
```

---

## 2. 组合（嵌套）状态

```plantuml
@startuml
[*] --> NotShooting

state NotShooting {
  [*] --> Idle
  Idle --> Configuring : EvConfig
  Configuring --> Idle : EvDone
}

state Configuring {
  [*] --> Edit
  Edit --> Preview : 预览
  Preview --> Edit : 返回
}
@enduml
```

---

## 3. 历史状态 [H] / [H*]

```plantuml
@startuml
state 工作中 {
  [*] --> 编辑
  编辑 --> 预览 : 预览
  预览 --> 编辑 : 返回
}

暂停 --> 工作中[H] : 恢复到上次子状态
深度恢复 --> 工作中[H*] : 深层历史
@enduml
```

`[H]`：同一层历史；`[H*]`：深层历史。

---

## 4. 并发区域 -- / ||

```plantuml
@startuml
[*] --> Active
state Active {
  [*] --> NumOff
  NumOff --> NumOn : NumLock
  NumOn --> NumOff : NumLock
  --
  [*] --> CapsOff
  CapsOff --> CapsOn : CapsLock
  CapsOn --> CapsOff : CapsLock
}
@enduml
```

`--` 水平分隔并发；`||` 竖直分隔。

---

## 5. fork / join / choice

```plantuml
@startuml
state fork1 <<fork>>
state join1 <<join>>
state c <<choice>>

[*] --> fork1
fork1 --> A
fork1 --> B
A --> join1
B --> join1
join1 --> [*]

Idle --> c
c --> Small : [id <= 10]
c --> Large : [id > 10]
@enduml
```

---

## 6. 完整样例：订单状态

```plantuml
@startuml
title 订单状态
[*] --> 待支付 : 创建
待支付 --> 已支付 : 支付成功
待支付 --> 已取消 : 超时/用户取消
已支付 --> 已发货 : 仓库出库
已发货 --> 已完成 : 签收
已支付 --> 退款中 : 申请退款
退款中 --> 已取消 : 退款完成
已完成 --> [*]
已取消 --> [*]

note right of 待支付
  通常 15～30 分钟关闭
end note
@enduml
```

## 7. 注释与链路上的 note

```plantuml
@startuml
[*] --> Open
Open --> Closed : close
note on link
  需要管理员权限
end note
@enduml
```

## 8. 练习

1. 画「工单」：新建 → 处理中 → 待确认 → 关闭，含驳回回退。  
2. 给「处理中」做成组合状态，内含「调研 / 开发 / 联调」。  
3. 解释清楚：订单取消可以从哪些状态到达？用图证明。
