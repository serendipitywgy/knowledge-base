# 06 · 对象图（Object）

← [[05-类图]] · [[PlantUML从入门到精通|目录]] · 下一章 → [[07-状态图]]

官方：https://plantuml.com/zh/object-diagram

对象图是类图在**某一时刻的实例快照**：具体对象、字段当前值、实例之间的链接。适合讲清「此刻系统里到底有什么」。

---

## 1. 何时用对象图

| 场景 | 用对象图 | 用类图 |
|------|----------|--------|
| 给新人讲一笔真实订单长什么样 | ✅ | 只看到类型 |
| 定义领域规则、多重性 | 辅助例子 | ✅ 主图 |
| 排障：内存/会话里对象图 | ✅ | 不直观 |

口诀：**类图画规则，对象图画例子。**

---

## 2. 声明对象

```plantuml
@startuml
object firstObject
object "我的第二个对象" as o2
object user {
  name = "Dummy"
  id = 123
}
@enduml
```

也可用冒号逐条写属性：

```plantuml
@startuml
object user
user : name = "Dummy"
user : id = 123
@enduml
```

---

## 3. 对象之间的关系

符号与类图一致，但语义落在**实例**上：

| 类型 | 符号 |
|------|------|
| 扩展 | `<|--` |
| 实现 | `<|..` |
| 组合 | `*--` |
| 聚合 | `o--` |
| 关联 / 依赖 | `-->` / `..>` |

```plantuml
@startuml
object Object01
object Object02
object Object03
object Object04
object Object05
object Object06
object Object07
object Object08

Object01 <|-- Object02
Object03 *-- Object04
Object05 o-- "4" Object06
Object07 .. Object08 : some labels
@enduml
```

虚线关系：把 `--` 换成 `..`。标签用 `: 文字`，多重性用 `"1"` / `"*"` 写在两端。

---

## 4. 关联类式：diamond

多个对象汇聚到一个菱形关联：

```plantuml
@startuml
object o1
object o2
diamond dia
object o3
o1 --> dia
o2 --> dia
dia --> o3
@enduml
```

## 5. 业务样例：一笔订单快照

```plantuml
@startuml
title 某一刻的订单实例

object customer {
  id = 1001
  name = "Alice"
}

object order7788 {
  id = "O-7788"
  status = "PAID"
  total = 199.00
}

object item1 {
  sku = "BOOK-1"
  qty = 2
  price = 49.50
}

object item2 {
  sku = "PEN-3"
  qty = 1
  price = 100.00
}

object pay1 {
  channel = "wechat"
  amount = 199.00
}

customer "1" --> "1" order7788 : places
order7788 *-- item1
order7788 *-- item2
order7788 --> pay1 : paid by

note right of order7788
  快照时间：支付成功后
  库存已扣减（未画出）
end note
@enduml
```

## 6. map：表 / 关联数组

```plantuml
@startuml
map CapitalCity {
  UK => London
  USA => Washington
  Germany => Berlin
}
@enduml
```

与对象连线、指向 map 中某一项：

```plantuml
@startuml
object London
object Washington
object Berlin
object NewYork

map CapitalCity {
  UK *-> London
  USA *--> Washington
  Germany *---> Berlin
}
NewYork --> CapitalCity::USA
@enduml
```

也可用 map 做简易 PERT 节点（任务表）：

```plantuml
@startuml
left to right direction
title PERT 示意
map Kick.Off {
}
map task.1 {
  Start => End
}
map task.2 {
  Start => End
}
Kick.Off --> task.1
Kick.Off --> task.2
task.1 --> task.2
@enduml
```

## 8. 与类图并排

```plantuml
@startuml
class Customer
class Order

object alice
object order7788

Customer .. alice
Order .. order7788
alice --> order7788
@enduml
```
